📜 Description du jeu

Il s'agit d'un jeu d'arcade mettant en scène un vaisseau spatial et des murs de météorites.
Le jeu commence par un affichage pour donner au joueur le temps de se préparer. Dès le début du jeu le vaisseau et les premiers murs apparaissent. La musique de fond commence.
Le jeu est mis en pause grâce à la touche "p" et est repris grâce à cette même touche. Pour lancer un missile c'est avec la touche "Espace". Si le missile touche un mur, les deux disparaissent avec un bruit de collision. Un message d'encouragement s'affiche. Initialement le joueur a droit à 4 missiles mais grâce à un but il peut en obtenir deux de plus dans ce cas un bref son de victoire retentit. Un message s'affiche rapidement pour l'en informer.
En cas de collision du vaisseau avec un mur c'est un échec et le jeu s'arrête. On a alors un message indiquant la défaite suivi d'un son dans le même ton.
En cas de victoire c'est-à-dire collision du vaisseau avec la ligne d'arrivée un message de victoire indiquant le temps mis en secondes s'affiche suivi d'un son de victoire.
2 secondes après la fin du jeu que ce soit par une victoire ou une défaite le jeu se ferme.

💻 Description du code

Le code est découpé en 2 modules.

  • Le module world constitué de world.h et de world.c. On y a regroupé les fonctions et les structures en rapport avec le monde du jeu(le sprite,les murs,...)
  • Le module graphiques constitué de graphiques.h et graphiques.c. On y a regroupé les fonctions et les structures en rapport avec l'affichage (les textures,les sons,...)
On a aussi constantes.h où on a mis toutes les constantes utilisées dans le jeu.
Et pour finir main.c c'est le fichier principal qui inclut tous les autres. On y trouve la boucle du jeu où on appelle toutes les fonctions.

🎮 Fonctionnalités développées

  • Déplacement du joueur avec les touches directionnelles
  • Ajout de murs supplémentaires pour allonger le parcours
  • Musique de fond
  • Effets sonores (défaite, victoire, collision)
  • Affichage de messages (défaite, victoire, encouragements)
  • Possibilité de mettre en pause le jeu
  • Ajouts de missiles
  • Ajout d'un but permettant d'obtenir des missiles bonus

🐞 Bugs connus

  • Collisions parfois imprécises
  • Affichage de messages saccadé à certains moments
  • Lancement de missile entrainant quelques secondes de bug
