## À propos

Passionné par la programmation depuis ma L1, j'ai développé des projets allant du jeu vidéo en C avec SDL2 jusqu'à des algorithmes d'IA (Minimax, Alpha-Beta, Monte-Carlo) en passant par des applications desktop JavaFX avec architecture MVVM et persistance SQLite. J'aime comprendre comment les choses fonctionnent en profondeur — c'est pourquoi j'ai implémenté les mêmes structures de données en tableau, liste chaînée et hashmap pour les comparer directement.
