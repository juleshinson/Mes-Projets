# 🍽️ Recettes Africaines — Site de cuisine HTML/CSS

Projet universitaire réalisé en **L1 Informatique** à l'Université de Lorraine (décembre 2024).  
Développé en binôme avec **Mahsoun**.

Site web statique présentant des recettes de cuisine africaine — plats marocains et béninois — avec une interface visuelle soignée et une navigation par catégorie.

---

## Contenu

**4 recettes détaillées :**

| Plat | Origine |
|------|---------|
| Tajine traditionnel | Maroc |
| Couscous | Maroc |
| Brochettes d'alloco | Bénin |
| Atassi (riz aux haricots blancs) | Bénin |

Chaque page recette présente les ingrédients avec photos et les étapes de préparation.

## Structure du projet

```

├── betaprincipale/
│   ├── index.html          # Page d'accueil (catalogue des recettes)

│   ├── recette_couscous.html

    ├── Images_principales/ # Photos des plats pour l'accueil
```

---

## Lancement

Aucune dépendance requise — le site fonctionne directement dans un navigateur.

```bash
# Ouvrir simplement dans un navigateur
open projet/betaprincipale/index.html
# ou double-clic sur le fichier index.html
```

---

## Aperçu

La page d'accueil présente les plats en deux colonnes (marocains / béninois) avec photos cliquables. Chaque recette s'ouvre sur une page dédiée avec ingrédients illustrés et étapes de préparation.
